# Beauty Mix — Советы для приложения Люмэри
## Инструкция для Кати

Ниже 120 советов, собранных из профессиональных источников (памятки мастеров, косметологические клиники, обучающие центры). Каждый совет проверен по открытым рекомендациям практикующих специалистов.

**Что нужно сделать:**
1. Прочитать каждый совет
2. В колонке «Одобрено» поставить ✅ или ❌
3. Если нужно исправить формулировку — написать свой вариант в колонке «Правка»
4. Если есть свои советы — добавить в конец таблицы

**Колонка «Сезон»:** all = круглый год, winter = зима, summer = лето, spring = весна, autumn = осень. Если совет актуален только в определённый сезон — исправь.
